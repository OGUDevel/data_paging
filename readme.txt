+WHAT IS IT FOR ?: 

Taking all datas from your database and printing them on just a page is simple for you, but for your server and guests, it takes a long time.To avoid this, data are sorted and paging according to your wishes. Each guest works with data which is shown in instant page.

+HOW IS IT USE ?:

Before uploading data_paging.php file to your web server, open this file with notepad. Follow the instructions and write your database connection code to first php line. Then you can use it after uploading it to your web server.

+PROPERTIES:

-You can define how many records will be shown in each page.
-It hasn't a design, you can edit it for yourself.
-We wrote our best security protect, because we used GET method to take page numbers.
-It has back, forward and pages links as a number.
-The opened page's link is not active.
